# 速查表

## 科斯式第一反应

| 看到的问题 | 先问什么 |
|---|---|
| 为什么有企业 | 使用市场的成本是什么？内部组织的成本又是什么？ |
| 是否应政府干预 | 现实政府方案的运行成本是多少？ |
| 外部性损害 | 这是哪两种活动之间的相互冲突？ |
| 产权争议 | 哪种权利界定使高价值交易更容易发生？ |
| 公共品 | 是否真的无法收费？历史上如何供给？ |
| 奇怪商业实践 | 它是否节约了交易费用，而不只是垄断？ |

## 制度选择矩阵

| 安排 | 优点 | 主要成本 | 适用情境 |
|---|---|---|---|
| 现货市场 | 灵活、价格清晰 | 搜寻、谈判、质量和执行成本 | 标准化、低不确定性交易 |
| 长期合同 | 降低重复谈判 | 合同不完全、适应成本 | 重复交易、有专用投资 |
| 企业内部化 | 降低市场交易费用 | 管理、激励和信息成本 | 交易频繁、合同难写 |
| 法院裁判 | 明确权利、解决争端 | 诉讼、证据和执行成本 | 权利冲突、协商困难 |
| 政府管制 | 可处理多人协调 | 信息、行政、政治成本 | 交易费用极高或风险巨大 |
| 混合制度 | 可组合优势 | 设计复杂 | 公共服务、平台、特许经营 |

## 社会成本分析步骤

1. 定义冲突的两种或多种活动。
2. 不用“加害者/受害者”过早封口，先识别相互性。
3. 列出可能权利配置。
4. 判断各配置下交易是否可行。
5. 比较市场协商、法院、税收、管制、企业内部化等制度成本。
6. 选择总价值较高且执行可行的安排。

## 企业边界判断

| 如果... | 倾向 |
|---|---|
| 交易标准化、供应商多、质量易验证 | 市场购买 |
| 关系长期、需求可预期但需适应 | 长期合同 |
| 资产专用性强、机会主义风险高 | 内部化或强治理合同 |
| 内部管理错误和官僚成本高 | 外包或拆分 |
| 活动组合产生信息和协调优势 | 企业内组织 |

## 常见误读纠正

- 科斯定理不是“市场总能解决外部性”。
- 零交易费用是分析基准，不是现实假设。
- 产权界定不只影响分配，在正交易费用下也影响效率。
- 公共品不自动推出政府直接生产。
- 市场失败不自动推出政府成功。
- 企业不是对市场的背离，而是市场交易费用下的替代安排。
